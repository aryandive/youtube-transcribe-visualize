# YouTube Transcriber & Visualizer 🎥🧠

A high-performance, modern web application designed to turn long YouTube videos into structured, actionable insights. Extract transcripts, generate executive summaries, and visualize concepts through AI-generated mind maps and flowcharts.

## 🌟 Core Philosophy
**Frictionless First.** Users can extract transcripts and view basic visualizations immediately without creating an account. Strategic "conversion walls" are placed on premium actions (saving, exporting, deep zooming) to capture sign-ups and demonstrate immediate value.

## 🚀 Features
* **Instant Transcription:** Extract accurate captions directly from YouTube URLs.
* **AI Summarization:** Get executive summaries, bullet points, and key action items.
* **Mermaid.js Visualizations:** Automatically generate interactive Mind Maps and Flow Charts from video concepts.
* **Frictionless Guest Mode:** Try the core tools before hitting an authentication wall.
* **Premium Workspaces (Coming Soon):** Save history, export PDFs/Diagrams, and process longer videos.

## 🛠️ Tech Stack
This project is built with a focus on speed, type safety, and scalability.

**Frontend Core:**
* **Framework:** [React 18](https://react.dev/)
* **Build Tool:** [Vite](https://vitejs.dev/) (with SWC for lightning-fast compilation)
* **Language:** [TypeScript](https://www.typescriptlang.org/)

**Styling & UI:**
* **CSS Framework:** [Tailwind CSS](https://tailwindcss.com/)
* **Component Library:** [Shadcn/UI](https://ui.shadcn.com/) (Radix UI primitives)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Diagrams:** [Mermaid.js](https://mermaid.js.org/)

**State Management & Data Fetching:**
* **Server State:** [TanStack Query](https://tanstack.com/query/latest) (React Query)
* **Client State:** [Zustand](https://github.com/pmndrs/zustand)
* **Routing:** [React Router v6](https://reactrouter.com/)

**Infrastructure (Roadmap):**
* **Auth & Database:** Supabase
* **Payments:** Razorpay
* **Deployment:** Vercel
